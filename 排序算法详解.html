排序算法详解

🎯 核心思想

排序算法是将一组数据按照特定顺序（通常是升序或降序）重新排列的算法。排序是计算机科学中最基本且重要的操作之一，几乎所有的应用程序都会用到排序算法。

基本概念

核心特征：

  • 🔹 稳定性 ：相等元素的相对位置是否保持不变

  • 🔹 原地性 ：是否需要额外的存储空间

  • 🔹 时间复杂度 ：算法执行时间与数据规模的关系

  • 🔹 适应性 ：对不同数据分布的敏感程度

📊 经典算法分析

算法1：冒泡排序

问题描述： 重复地遍历要排序的数列，一次比较两个元素，如果顺序错误就交换，直到没有任何需要交换的元素。

核心思想： 每次遍历都会将最大（或最小）元素"冒泡"到正确位置

图解演示

算法步骤：

  1. 比较相邻的两个元素

  2. 如果前面的元素大于后面的元素，交换它们

  3. 对每一对相邻元素做同样的工作

  4. 每轮结束后，最大的元素会被交换到最后

  5. 重复以上步骤，直到没有任何交换

算法2：快速排序

问题描述： 使用分治策略，选择一个基准元素，将数组分为两部分，然后递归排序。

核心思想： 分治 + 原地分区

图解演示

算法步骤：

  1. 选择一个基准元素（pivot）

  2. 将数组分区，使得小于pivot的在左边，大于pivot的在右边

  3. 递归地对左右两部分进行快速排序

  4. 合并结果（实际上不需要合并，因为是原地排序）

算法3：归并排序

问题描述： 将数组分成两半，分别排序，然后合并两个有序数组。

核心思想： 分治 + 合并有序数组

图解演示

算法步骤：

  1. 将数组不断分割，直到只有一个元素

  2. 将相邻的有序子数组合并成更大的有序数组

  3. 重复合并过程，直到整个数组有序

算法4：堆排序

问题描述： 利用堆这种数据结构进行排序，首先建立最大堆，然后重复提取最大元素。

核心思想： 堆的性质 + 原地排序

图解演示

算法步骤：

  1. 将待排序数组构建成最大堆

  2. 将堆顶元素（最大值）与末尾元素交换

  3. 将剩余n-1个元素重新构建成最大堆

  4. 重复步骤2-3，直到堆的大小为1

🔧 算法选择策略

根据数据特征选择

1. 数据规模

小规模(n < 50): 插入排序 中规模(50 ≤ n ≤ 10000): 快速排序 大规模(n > 10000): 归并排序或堆排序

2. 数据分布

基本有序: 插入排序、冒泡排序 随机分布: 快速排序 逆序: 选择排序、堆排序 重复元素多: 三路快排

3. 内存限制

内存充足: 归并排序 内存受限: 快速排序、堆排序 极度受限: 插入排序、选择排序

4. 稳定性要求

需要稳定: 归并排序、插入排序、冒泡排序 不需要稳定: 快速排序、选择排序、堆排序

混合排序策略

1. 内省排序(Introsort)

结合快排、堆排和插入排序 大数据用快排，递归深度过深时转堆排 小数据用插入排序

2. TimSort

Python和Java的默认排序算法 结合归并排序和插入排序 针对真实数据优化

📈 实际应用场景

系统级排序

1. 数据库排序

  • 外部排序 ：数据量超过内存

  • 索引排序 ：维护有序索引

  • 查询优化 ：ORDER BY语句

2. 文件系统

  • 目录排序 ：按名称、时间、大小排序

  • 日志排序 ：按时间戳排序

  • 备份排序 ：按优先级排序

应用级排序

1. 搜索引擎

  • 相关性排序 ：按匹配度排序搜索结果

  • 网页排名 ：PageRank算法

  • 实时排序 ：新闻、股价等

2. 电商平台

  • 商品排序 ：按价格、销量、评分排序

  • 推荐排序 ：个性化推荐列表

  • 库存排序 ：按需求量排序

💡 解题模板

快速排序模板

def quick_sort(arr, low, high): if low < high: pi = partition(arr, low, high) quick_sort(arr, low, pi - 1) quick_sort(arr, pi + 1, high) def partition(arr, low, high): pivot = arr[high] i = low - 1 for j in range(low, high): if arr[j] <= pivot: i += 1 arr[i], arr[j] = arr[j], arr[i] arr[i + 1], arr[high] = arr[high], arr[i + 1] return i + 1

归并排序模板

def merge_sort(arr): if len(arr) > 1: mid = len(arr) // 2 left = arr[:mid] right = arr[mid:] merge_sort(left) merge_sort(right) merge(arr, left, right) def merge(arr, left, right): i = j = k = 0 while i < len(left) and j < len(right): if left[i] <= right[j]: arr[k] = left[i] i += 1 else: arr[k] = right[j] j += 1 k += 1 while i < len(left): arr[k] = left[i] i += 1 k += 1 while j < len(right): arr[k] = right[j] j += 1 k += 1

堆排序模板

def heap_sort(arr): n = len(arr) # 建立最大堆 for i in range(n // 2 - 1, -1, -1): heapify(arr, n, i) # 提取元素 for i in range(n - 1, 0, -1): arr[0], arr[i] = arr[i], arr[0] heapify(arr, i, 0) def heapify(arr, n, i): largest = i left = 2 * i + 1 right = 2 * i + 2 if left < n and arr[left] > arr[largest]: largest = left if right < n and arr[right] > arr[largest]: largest = right if largest != i: arr[i], arr[largest] = arr[largest], arr[i] heapify(arr, n, largest)
