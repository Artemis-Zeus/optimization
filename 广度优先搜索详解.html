广度优先搜索详解

🎯 核心思想

广度优先搜索(Breadth-First Search, BFS)是一种用于遍历或搜索树或图的算法。BFS从起始节点开始，首先访问起始节点的所有邻居节点，然后再访问这些邻居节点的邻居节点，以此类推，按层次逐步向外扩展。

基本原理

核心特征：

  • 🔹 层次遍历 ：按照距离起点的远近逐层访问

  • 🔹 队列结构 ：使用FIFO队列管理待访问节点

  • 🔹 最短路径 ：在无权图中能找到最短路径

  • 🔹 完整性 ：能够访问到所有可达节点

🧠 算法思维模式

BFS实现方式

1. 标准队列实现

特征： 使用队列保证先进先出的访问顺序

queue = [起始节点] visited = set() while queue: current = queue.popleft() if current in visited: continue visited.add(current) 处理当前节点 for 邻居 in current的邻居: if 邻居 not in visited: queue.append(邻居)

2. 层次遍历实现

特征： 明确区分每一层，便于记录层数或距离

queue = [起始节点] visited = set([起始节点]) level = 0 while queue: size = len(queue) for i in range(size): current = queue.popleft() 处理当前节点 for 邻居 in current的邻居: if 邻居 not in visited: visited.add(邻居) queue.append(邻居) level += 1

📊 经典题目分析

题目1：二叉树的层序遍历

问题描述： 给你一个二叉树，请你返回其按层序遍历得到的节点值。（即逐层地，从左到右访问所有节点）

BFS策略： 使用队列按层遍历，记录每层的节点

图解演示

解题思路：

  1. 将根节点入队

  2. 记录当前层的节点数量

  3. 依次处理当前层的所有节点

  4. 将下一层的节点加入队列

  5. 重复直到队列为空

题目2：最短路径问题

问题描述： 在一个m×n的二维网格中，从起点(0,0)到终点(m-1,n-1)的最短路径长度，只能向上下左右四个方向移动。

BFS策略： 从起点开始逐层扩展，第一次到达终点时即为最短路径

图解演示

解题思路：

  1. 从起点开始，距离为0

  2. 将起点加入队列

  3. 扩展当前层的所有节点

  4. 为下一层节点的距离+1

  5. 第一次到达终点时即为最短距离

题目3：单词接龙

问题描述： 给定两个单词（beginWord和endWord），和一个字典的单词列表，找到从beginWord到endWord的最短转换序列的长度。每次转换只能改变一个字母，且中间的单词必须是字典中的单词。

BFS策略： 将每个单词看作节点，相差一个字母的单词之间有边，求最短路径

图解演示

解题思路：

  1. 将起始单词加入队列，步数为1

  2. 对当前单词的每个位置尝试26个字母

  3. 如果生成的单词在字典中且未访问过，加入队列

  4. 记录每个单词的转换步数

  5. 第一次遇到目标单词时返回步数

题目4：最小基因变化

问题描述： 一条基因序列由一个带有8个字符的字符串表示，其中每个字符都属于"A","C","G","T"中的任意一个。要从start变化到end，需要一系列的变化过程，每次变化只能改变一个字符，且变化后的基因序列必须在bank中。

BFS策略： 每次变化看作一步，求最少变化次数

图解演示

解题思路：

  1. 将起始基因序列加入队列

  2. 对当前序列的每个位置尝试A,C,G,T四种字符

  3. 如果变化后的序列在bank中且未访问，加入队列

  4. 记录变化步数，第一次到达目标序列时返回步数

🔧 BFS设计技巧

队列管理策略

1. 基础队列

from collections import deque queue = deque([start]) visited = set([start]) while queue: current = queue.popleft() 处理当前节点 for neighbor in get_neighbors(current): if neighbor not in visited: visited.add(neighbor) queue.append(neighbor)

2. 层次队列

queue = deque([start]) visited = set([start]) level = 0 while queue: size = len(queue) for _ in range(size): current = queue.popleft() 处理当前层节点 for neighbor in get_neighbors(current): if neighbor not in visited: visited.add(neighbor) queue.append(neighbor) level += 1

3. 带距离信息的队列

queue = deque([(start, 0)]) # (节点, 距离) visited = set([start]) while queue: current, dist = queue.popleft() if current == target: return dist for neighbor in get_neighbors(current): if neighbor not in visited: visited.add(neighbor) queue.append((neighbor, dist + 1))

优化技巧

1. 双向BFS

当起点和终点都已知时，可以从两端同时搜索

从起点和终点同时开始BFS 当两个搜索相遇时停止 搜索深度减半，效率提升

2. 启发式BFS (A*)

结合启发式函数指导搜索方向

使用优先队列代替普通队列 根据f(n) = g(n) + h(n)排序 g(n)为实际距离，h(n)为启发式估计

3. 状态压缩

对于状态空间较大的问题，压缩状态表示

使用位运算压缩状态 哈希化复杂状态 减少内存使用

📈 复杂度分析

时间复杂度

  • 树结构BFS ：O(V)，V为节点数

  • 图结构BFS ：O(V + E)，E为边数

  • 网格BFS ：O(m × n)，m和n为网格尺寸

  • 状态空间BFS ：O(b^d)，b为分支因子，d为深度

空间复杂度

  • 队列存储 ：O(V)，最坏情况下队列包含所有节点

  • 访问标记 ：O(V)，记录已访问节点

  • 距离数组 ：O(V)，记录每个节点的距离

💡 解题模板

基础BFS模板

from collections import deque def bfs(start): queue = deque([start]) visited = set([start]) while queue: current = queue.popleft() # 处理当前节点 process(current) # 检查是否为目标 if is_target(current): return True # 扩展邻居节点 for neighbor in get_neighbors(current): if neighbor not in visited: visited.add(neighbor) queue.append(neighbor) return False

最短路径BFS模板

from collections import deque def shortest_path(start, target): queue = deque([(start, 0)]) visited = set([start]) while queue: current, distance = queue.popleft() if current == target: return distance for neighbor in get_neighbors(current): if neighbor not in visited: visited.add(neighbor) queue.append((neighbor, distance + 1)) return -1 # 无法到达

层次遍历BFS模板

from collections import deque def level_order_traversal(root): if not root: return [] result = [] queue = deque([root]) while queue: level_size = len(queue) level_nodes = [] for _ in range(level_size): node = queue.popleft() level_nodes.append(node.val) if node.left: queue.append(node.left) if node.right: queue.append(node.right) result.append(level_nodes) return result
