二分查找算法详解

🎯 核心思想

二分查找(Binary Search)是一种在有序数组中查找特定元素的搜索算法。通过重复将搜索区间分成两半，并根据中间元素与目标值的比较结果来确定下一步搜索的方向，从而将时间复杂度降低到O(log n)。

基本原理

核心特征：

  • 🔹 有序前提 ：数组必须是有序的

  • 🔹 分治思想 ：每次排除一半的搜索空间

  • 🔹 对数时间 ：时间复杂度为O(log n)

  • 🔹 边界处理 ：正确处理边界条件是关键

🧠 算法思维模式

二分查找类型分类

1. 基础二分查找

特征： 查找精确值，找到即返回

目标：在数组中查找target 返回：target的索引，不存在返回-1

2. 左边界二分查找

特征： 查找第一个大于等于target的位置

目标：找到第一个 >= target 的位置 返回：插入位置或第一次出现的位置

3. 右边界二分查找

特征： 查找最后一个小于等于target的位置

目标：找到最后一个 <= target 的位置 返回：最后一次出现的位置

4. 峰值查找

特征： 在无序数组中查找局部极值

目标：找到比相邻元素都大的峰值 应用：旋转数组、山脉数组

📊 经典题目分析

题目1：基础二分查找

问题描述： 给定一个n个元素有序的（升序）整型数组nums和一个目标值target，写一个函数搜索nums中的target，如果目标值存在返回下标，否则返回-1。

二分查找策略： 标准二分查找模板

图解演示

解题思路：

  1. 初始化左右边界：left=0, right=n-1

  2. 计算中间位置：mid = left + (right-left)/2

  3. 比较nums[mid]与target

  4. 根据比较结果调整搜索边界

  5. 重复直到找到或搜索空间为空

题目2：搜索插入位置

问题描述： 给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

二分查找策略： 左边界二分查找

图解演示

解题思路：

  1. 使用左边界二分查找模板

  2. 找到第一个大于等于target的位置

  3. 如果找到相等元素，返回该位置

  4. 如果没找到，返回应该插入的位置

题目3：搜索旋转排序数组

问题描述： 整数数组nums按升序排列，数组中的值互不相同。在传递给函数之前，nums在预先未知的某个下标k上进行了旋转。给你旋转后的数组nums和一个整数target，如果nums中存在这个目标值target，请你返回它的下标，否则返回-1。

二分查找策略： 判断有序部分，在有序部分进行二分查找

图解演示

解题思路：

  1. 旋转数组至少有一半是有序的

  2. 先判断哪一半是有序的

  3. 检查target是否在有序部分的范围内

  4. 在有序部分进行二分查找

  5. 否则在另一半继续搜索

题目4：寻找峰值

问题描述： 峰值元素是指其值大于左右相邻值的元素。给你一个整数数组nums，找到峰值元素并返回其索引。数组可能包含多个峰值，在这种情况下，返回任何一个峰值所在位置即可。

二分查找策略： 根据斜率方向进行二分查找

图解演示

解题思路：

  1. 数组两端可以视为负无穷

  2. 如果mid处于上升趋势，峰值在右边

  3. 如果mid处于下降趋势，峰值在左边或就是mid

  4. 每次都能排除一半搜索空间

🔧 二分查找设计技巧

边界处理策略

1. 闭区间写法 [left, right]

left = 0, right = n - 1 while left <= right: mid = left + (right - left) // 2 if nums[mid] == target: return mid elif nums[mid] < target: left = mid + 1 else: right = mid - 1 return -1

2. 左闭右开写法 [left, right)

left = 0, right = n while left < right: mid = left + (right - left) // 2 if nums[mid] < target: left = mid + 1 else: right = mid return left

3. 开区间写法 (left, right)

left = -1, right = n while left + 1 < right: mid = left + (right - left) // 2 if nums[mid] < target: left = mid else: right = mid return right

防止溢出技巧

中点计算

// 可能溢出 mid = (left + right) / 2 // 防溢出写法 mid = left + (right - left) / 2

边界更新

// 避免死循环 left = mid + 1 // 而不是 left = mid right = mid - 1 // 而不是 right = mid

📈 复杂度分析

时间复杂度

  • 基础二分查找 ：O(log n)

  • 变形二分查找 ：O(log n)

  • 二分答案 ：O(log(max-min) × 验证函数复杂度)

空间复杂度

  • 迭代实现 ：O(1)

  • 递归实现 ：O(log n)，递归栈空间

✅ 适用场景识别

适合二分查找的问题特征

  1. 📊 有序数组/序列

    • 严格有序或部分有序

    • 可以确定搜索方向

  2. 🎯 单调性问题

    • 答案具有单调性

    • 可以通过比较排除一半空间

  3. 🔍 查找边界问题

    • 查找第一个/最后一个满足条件的位置

    • 插入位置问题

  4. ⚖️ 二分答案

    • 答案在某个范围内

    • 可以验证答案的正确性

识别关键词

✅ "有序数组" ✅ "查找/搜索" ✅ "第一个/最后一个" ✅ "插入位置" ✅ "旋转/峰值" ✅ "单调性"

不适合二分查找的情况

❌ 完全无序的数据 ❌ 链表等非随机访问结构 ❌ 答案不具有单调性 ❌ 搜索空间无法有效分割

💡 解题模板

基础二分查找模板

def binary_search(nums, target): left, right = 0, len(nums) - 1 while left <= right: mid = left + (right - left) // 2 if nums[mid] == target: return mid elif nums[mid] < target: left = mid + 1 else: right = mid - 1 return -1

左边界查找模板

def left_bound(nums, target): left, right = 0, len(nums) while left < right: mid = left + (right - left) // 2 if nums[mid] < target: left = mid + 1 else: right = mid return left

右边界查找模板

def right_bound(nums, target): left, right = 0, len(nums) while left < right: mid = left + (right - left) // 2 if nums[mid] <= target: left = mid + 1 else: right = mid return left - 1

二分答案模板

def binary_answer(left, right): while left < right: mid = left + (right - left) // 2 if check(mid): # 检查mid是否满足条件 right = mid # 答案在左半部分 else: left = mid + 1 # 答案在右半部分 return left
